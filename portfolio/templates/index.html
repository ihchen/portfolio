MemPassGen

April 2017

This project was based off an idea I had to create a master password for my password manager. I wanted something easy to remember but difficult to crack, and ended up picking a memorable phrase to me and simply each character to a nearby character on the keyboard. This way, you can avoid actual words but not have to change your muscle memory too much. So I created MemPassGen, which maps your input to keys based on your finger placement on a qwerty keyboard.

You can view the source code here.

Technology:
  • AngularJS (1.6.4)
  • Bootstrap Grid System
  • Karma/Jasmine
  • External Javascript libraries:
    • zxcvbn

Soundulate

Spring 2016

Soundulate is an ear training program to help University of Puget Sound music students taking the Aural Skills course practice their ability to distinguish different kinds of chords and scales. This was my capstone project in which I collaborated with two other fellow computer science musicians to make. The application replaced an old music program that students detested but had to purchase for the class. Soundulate was being used while we were working on it and still is.
